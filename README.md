# Retro 2D Game

A modern retro 2D game built with [LÖVE](https://love2d.org/) and Lua.

## Overview
Modern retro game inspired by the aesthetic and atmosphere of *V/H/S* (the found-footage horror anthology series).

## Project Structure
```text
├── assets/          # Sprites, fonts, and media
├── states/          # State modules (title screen, options menu, etc.)
├── config.lua       # Input mappings and configuration data
├── main.lua         # Engine lifecycle hooks and state router
└── README.md
